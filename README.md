A high-performance image compression tool written in modern C++.

## Installation
To build and install the Image Compression Tool, follow these steps:
1. Clone the repository: `git clone https://github.com/your-username/ImageCompressionTool.git`
2. Navigate to the project directory: `cd ImageCompressionTool`
3. Create a build directory: `mkdir build`
4. Navigate to the build directory: `cd build`
5. Run CMake: `cmake..`
6. Build the project: `cmake --build.`
7. Install the project: `cmake --install.`
